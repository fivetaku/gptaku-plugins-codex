# MCP / App 카탈로그 (Codex)

> 스킬 생성 시 활용 가능한 외부 연동(MCP 서버 / Codex App) 목록. 전문가 관점이 아이디어 분석 시 적합한 연동을 추천하는 데 사용.

## 사용법

Phase B 전문가 관점이 아이디어를 분석할 때, 입력/출력/외부 연동 요구사항을 아래 카탈로그와 매칭한다. **API > MCP/App > 직접 구현** 우선순위를 적용한다(`references/api-mcp-integration.md`). 추천 시 사용자가 해당 연동을 설치·설정해야 함을 안내한다.
