# pumasi-codex

Codex marketplace package for the `pumasi` family.

Included skills:
- `pumasi` — Parallel coding orchestration for multi-module greenfield work.
- `pumasi-image` — Native image generation companion for posters, thumbnails, and other visual deliverables.

Structure:
- `skills/pumasi/` — Parallel build orchestration skill.
- `skills/pumasi-image/` — Native image-generation companion skill.
- `.codex-plugin/plugin.json` — Codex plugin manifest

Packaging notes:
- The plugin keeps `pumasi` and `pumasi-image` together as one product unit.
