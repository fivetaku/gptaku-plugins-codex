# insane-design-codex

Codex marketplace package for the curated `insane-design` workflow.

Included skills:
- `insane-design` — Extracts a real design system from a live URL into `design.md` and `report.ko.html`, or reuses the bundled corpus as a deterministic design brief.

Packaging notes:
- This package keeps the bundled `examples/` corpus because it is part of the product value.
- The packaged plugin includes only the Codex-native contract files for workflow, template, and shared rules.
