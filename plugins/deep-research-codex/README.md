# deep-research-codex

Codex marketplace package for the `deep-research` product unit.

Included skills:
- `deep-research` — Runs structured, citation-heavy research sessions with durable `RESEARCH/` state.
- `deep-research-query` — Turns vague research ideas into a schema-backed research brief.

Packaging notes:
- The main research workflow and query builder stay together because the original product shipped them as one research suite.
- Active docs are Codex-native and chat-first; there is no widget dependency in the packaged runtime.
